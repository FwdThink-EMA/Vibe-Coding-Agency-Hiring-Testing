# Submission Instructions

Place your final deliverables in this folder.

You must submit **one PDF per test**, with all explanations and diagrams embedded directly in the PDF (no external images or links).

```
submission/
  cloud-test.pdf
  security-test.pdf
  application-test.pdf
```

- Do **not** change the filenames.
- Do **not** add additional folders or files unless a specific test requires it.
- Do **not** link to Google Docs, Figma, Lucidchart, or external hosting. All visuals must be included directly in the PDFs.

When finished, commit and push this folder to your fork and send us the repository link.
